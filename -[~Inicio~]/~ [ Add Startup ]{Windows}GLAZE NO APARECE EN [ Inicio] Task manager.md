1- SOLUCION: Carpeta de Inicio de Windows (Más simple)

Presiona Win + R y escribe: shell:startup
Se abrirá la carpeta de inicio
Crea un acceso directo de GlazeWM ahí:

Busca el ejecutable de GlazeWM (probablemente en C:\Program Files\GlazeWM\ o donde lo instalaste)
Clic derecho en glazewm.exe → "Crear acceso directo"
Mueve el acceso directo a la carpeta de inicio


~ Luego Windows + Shift + Esc > Task Manager > Inicio:
 Habilitar glaze
